# Change Log

All notable changes to the "add-gitignore" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.3] - 2026-01-30

- Added support for Git worktrees when adding to `.git/info/exclude`.
- Improved `.git` directory discovery by searching upwards from the selected file.
- Clarified error messages when a Git repository cannot be found.

## [0.0.2] - 2026-01-28

- Improved extension compatibility.
- Updated metadata and engine versions.

## [0.0.1] - 2026-01-27

- Initial release.
