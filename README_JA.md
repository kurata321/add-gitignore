# add-gitignore

VS Codeのエクスプローラーコンテキストメニューから、ファイルを `.gitignore` や `.git/info/exclude` に素早く追加できます。

## 特徴

- **Git Ignoreサブメニュー**: コンテキストメニュー内の「Git Ignore」サブメニューに各コマンドがまとめられ、メニューがすっきりしました。
- **.gitignore に追加**: 対象の相対パスをワークスペースルートの `.gitignore` に追加します。
- **.git/info/exclude に追加**: 共有の `.gitignore` を変更せずにローカルでファイルを無視します。
- **重複防止**: 追加する前に、対象の無視ファイルに対象のパスがすでに存在するかどうかを確認します。
- **自動フォーマット**: パスが新しい行に正しく追加されるようにします。

## 使い方

1. エクスプローラービューを開きます。
2. 無視したいファイルまたはフォルダーを右クリックします。
3. **Git Ignore** を選択し、**Add to .gitignore** または **Add to .git/info/exclude** を選択します。

## 要件

- ワークスペースが開いている必要があります。
- 「.git/info/exclude に追加」を使用する場合、ワークスペースルートに `.git` ディレクトリが存在する必要があります。
